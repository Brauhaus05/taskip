# Taskip

[![Live Demo](https://img.shields.io/badge/Live_Demo-brauhaus05.github.io%2Ftaskip-40694E?style=for-the-badge&logo=github)](https://brauhaus05.github.io/taskip/)

A tenant property-management prototype (rent, maintenance, messages). Fully
client-side — no backend, no database. State persists in the browser via
localStorage; use the in-app **Reset demo data** button to restore the sample content.

**Live demo:** https://brauhaus05.github.io/taskip/

## Run
- `npm install`
- `npm run dev` — open the printed local URL
- `npm test` — run the test suite
- `npm run build` — production build

Converted from the original `prototype.html` design export (kept in the repo for reference).
